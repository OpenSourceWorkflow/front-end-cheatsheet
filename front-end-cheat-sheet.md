# Front End Cheat Sheet

* Legend: `<>` required, `[]` optional

## Node

### Search the registry for package

```bash
$ npm search <term>
```

### Install package

```bash
$ npm install [-g] <package-name>
```

### List packages

```bash
$ npm ls [package-name]
```

### Go to issue page of package

```bash
$ npm bugs <package-name>
```

### List outdated packages

```bash
$ npm outdated
```

### Update packages

```bash
$ npm update [package-name]
```

### Remove packages

```bash
$ npm rm <package-name>
```
